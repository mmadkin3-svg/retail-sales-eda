# Retail Sales EDA

**Tools:** BigQuery SQL · Tableau Public
**Dataset:** Mohammad Talib · Kaggle Retail Sales Dataset (1,000 transactions)

## Business Question
Which product categories and customer segments drive the most revenue,
and are there seasonal patterns we can use for inventory planning?

## Key Findings
- Electronics generates 38% of revenue despite 34% of transactions
- Revenue peaks in May; February dip is addressable with targeted campaigns
- The 35–44 age cohort has the highest average order value ($521)
